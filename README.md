## Problem

Uzun illərdir ki, gündəlik iş axtaranlar sərt hava şəraitində küçələrdə və bazarların qarşısında saatlarla iş gözləmək məcburiyyətində qalırlar. El arasında "qul bazarı" adlandırılan bu ənənəvi proses həm işçilər üçün çətindir, həm də səmərəsizdir.

Problem yalnız bir tərəfli deyil:

- **İşçi tərəfi** — İş tapmaq üçün fiziki olaraq müəyyən məkanlarda gözləmək məcburiyyəti, qeyri-müəyyənlik və güvənsizlik.
- **İşəgötürən tərəfi** — Evinə, həyətinə və ya obyektinə cəlb etdiyi şəxsin peşəkarlığına dair heç bir təminatın olmaması və tanışlıq vasitəsilə işçi axtarmaq məcburiyyəti.

Hazırda insanlar müxtəlif elan saytları və sosial şəbəkələr vasitəsilə çarə axtarsalar da, hər iki tərəfi birləşdirən, **güvən verən bir rəqəmsal körpü** mövcud deyil.

---

## Həll

**1ishvar.az** — günlük əmək bazarını rəqəmsallaşdıran, işçi ilə işəgötürəni şəffaf və etibarlı şəkildə görüşdürən platformadır.

### Necə işləyir?

| Addım | İşçi | İşəgötürən |
|-------|------|------------|
| **1** | Profil yaradır — ixtisasını və günlük xidmət haqqını qeyd edir | Ehtiyacı olan xidmət üzrə elan yerləşdirir |
| **2** | Uyğun elanlara müraciət edir | Müraciətləri dəyərləndirir və ən uyğun namizədi seçir |
| **3** | İşi icra edir və nağd ödəniş alır | İşi qəbul edir və hesablaşır |
| **4** | İşəgötürən haqqında rəy bildirir | İşçini qiymətləndirir və rəy yazır |

> Platforma təfərrüatlı müraciət və ikitərəfli rəy sisteminə əsaslanaraq hər iki tərəf üçün maksimum şəffaflığı təmin edir.

### Ödəniş Sistemi

Ödəniş prosesi **birbaşa hesablaşma** modelindədir — işəgötürən xidmət haqqını işçiyə nağd şəkildə ödəyir. Günlük işçilər üçün əlçatanlığı qorumaq məqsədilə ilkin mərhələdə onlayn ödəniş və bank hesabı tələbi tətbiq edilmir.

### Rəy və Etimad Mexanizmi

- İş bitdikdən sonra **hər iki tərəf** bir-birini dəyərləndirir.
- Yalnız real iş əlaqəsi təsdiqlənmiş tərəflər rəy yaza bilər.
- Yüksək reytinq toplamaq istəyən hər kəs platforma daxilində fəaliyyət göstərməyə təşviq edilir.

## Gəlir Modeli

| Mənbə | Təfərrüat |
|-------|-----------|
| **Xidmət haqqı** | Platformadan iş tapan işçinin aylıq gəlirinin 1–2%-i |
| **İnvoys sistemi** | Aylıq hesab göndərilir, ödəniş üçün 10 gün müddət tanınır |
| **Hesab dondurma** | Öhdəliyini yerinə yetirməyən istifadəçilərin profili müvəqqəti dondurulur |
| **Reklam** | Platforma daxili hədəflənmiş reklam gəliri |
| **Sığorta tərəfdaşlığı** | Sığorta şirkətləri ilə pilot layihə: günlük işçi sığortasından əldə olunan komissiya |
